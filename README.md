# 🎵 Sunoprompt - Suno AI プロンプト生成ツール

Suno AIで**高品質な音楽**を作るための、プロフェッショナルなプロンプト生成Chrome拡張機能です。

## 🚀 主な特徴

- **🎨 豊富なテーマ**: 90種類以上のプリセットテーマ + カスタムテーマ対応
- **🎸 多彩なジャンル**: 150種類のジャンルから最大3つまで組み合わせ可能
- **🎵 詳細な音楽設定**: BPM、キー、言語比率を細かく調整
- **🎤 楽曲構成**: パート別の詳細設定で理想的な楽曲構成を作成
- **💾 自動保存**: 入力内容を自動保存・復元
- **🔒 安全性**: OpenAI APIキーを暗号化して安全に保存

## 📥 インストール方法

### 手動インストール（開発者モード）

1. **リポジトリをダウンロード**
   ```bash
   git clone https://github.com/usedhonda/sunoprompt.git
   ```

2. **Chrome拡張機能として読み込み**
   - Chrome ブラウザで `chrome://extensions/` を開く
   - 右上の「デベロッパーモード」をONにする
   - 「パッケージ化されていない拡張機能を読み込む」をクリック
   - ダウンロードした `extension` フォルダを選択

3. **ツールバーに追加**
   - 拡張機能アイコンがChromeツールバーに表示されます

## 🛠️ セットアップ

### 1. OpenAI APIキーを取得

1. [OpenAI Platform](https://platform.openai.com/) にアクセス
2. アカウント作成・ログイン
3. API Keys ページで新しいAPIキーを作成
4. キーをコピー（後で使用）

### 2. 拡張機能を設定

1. Chromeツールバーの拡張機能アイコンをクリック
2. 「OpenAI API Key」欄にコピーしたキーを貼り付け
3. モデルを選択（推奨: GPT-4.1 Mini）

## 📖 使い方ガイド

### 基本的な流れ

1. **📝 テーマを選択**
   - プリセットテーマから選択、または
   - カスタムテーマを自由記述
   - 両方の組み合わせも可能

2. **🎵 ジャンルを選択**
   - カテゴリーから最大3つまで選択
   - 複数選択で独創的なスタイルを作成

3. **⚙️ 音楽設定を調整**
   - **BPM**: テンポを60-200で設定
   - **キー**: メジャー/マイナーと音楽キーを選択
   - **言語比率**: 日本語と英語の混合比率

4. **🎼 楽曲構成をカスタマイズ**
   - テンプレートから選択、または
   - 各パート（Intro、Verse、Chorus等）を個別設定
   - エネルギーレベルとボーカルスタイルを調整

5. **🎯 プロンプト生成**
   - 「プロンプト生成」ボタンをクリック
   - AIが最適化されたプロンプトを自動生成

6. **📋 Suno AIで使用**
   - 生成されたプロンプトをコピー
   - Suno AIに貼り付けて音楽を生成

### 便利な機能

#### 🔄 自動保存・復元
- 入力内容は自動的に保存されます
- 拡張機能を開き直しても前回の設定が復元されます

#### 🎨 テーマの組み合わせ
```
例：プリセット「深夜の街」+ カスタム「雨音と街のネオンが織りなす孤独な物語」
→ より具体的で個性的なテーマが完成
```

#### 🎵 ジャンルの組み合わせ
```
例：「Lo-Fi Hip Hop」+「Jazz」+「Ambient」
→ リラックスした雰囲気のユニークなサウンド
```

#### 🐛 デバッグモード
- `Ctrl+Shift+D` で詳細ログ表示
- 開発者向けの高度な設定が利用可能

## 💡 使用のヒント

### 良いプロンプトを作るコツ

1. **具体的なテーマを選ぶ**
   - ❌ 「恋愛」
   - ✅ 「雨の日のカフェで偶然再会した初恋の人との会話」

2. **ジャンルを組み合わせる**
   - 1つより2-3個の組み合わせの方がユニーク
   - 相性の良い組み合わせを実験

3. **言語比率を調整**
   - 日本語歌詞：自動でひらがな化（音声合成精度向上）
   - 英語歌詞：グローバルアピール
   - 混合：両方の良さを活用

4. **楽曲構成を活用**
   - シンプル：短い曲、ループ重視
   - 詳細：ストーリー性のある長い曲

## ⚠️ トラブルシューティング

### よくある問題

**🔴 APIキーエラー**
- OpenAI APIキーが正しく入力されているか確認
- APIクレジットが残っているか確認

**🔴 拡張機能が動かない**
- Chrome拡張機能ページで有効になっているか確認
- ページをリロードして再試行

**🔴 プロンプト生成が失敗する**
- インターネット接続を確認
- APIキーの有効性を確認

## 🔒 プライバシー・セキュリティ

- **ローカル処理**: すべての処理はブラウザ内で実行
- **暗号化保存**: APIキーは暗号化してローカル保存
- **最小限通信**: OpenAI API以外には一切送信しません
- **透明性**: ソースコードは公開されています
