Sunoprompt Extension - プライバシーポリシー

日本語 English

📋 プライバシーポリシー（日本語）

1. 収集する情報

本拡張機能は以下の情報を収集します：

  • 音楽設定: テーマ、ジャンル、楽器、キー、BPM
  • APIキー: OpenAI API認証用（暗号化して保存）
  • 楽曲構成: パート別の詳細設定

2. 情報の保存と使用

  • 保存場所: Chromeブラウザ内のみ（localStorage/syncStorage）
  • 使用目的: 音楽プロンプト生成の品質向上
  • 暗号化: APIキーはBase64エンコードで保存
  • 外部送信: OpenAI APIへのプロンプト生成のみ

3. 第三者との共有

収集した情報は第三者と共有されません。OpenAI APIとの通信は音楽プロンプト生成の目的のみに使用されます。

4. データの削除

  • 完全削除: 拡張機能をアンインストール
  • 手動削除: Chrome設定からストレージをクリア
  • 自動削除: 設定画面の「データをクリア」ボタン

5. セキュリティ

APIキーは暗号化され、ブラウザ内のみに保存されます。外部サーバーには送信されません。

📋 Privacy Policy (English)

1. Information We Collect

This extension collects the following information:

  • Music Settings: Theme, genre, instruments, key, BPM
  • API Keys: OpenAI API authentication (encrypted storage)
  • Song Structure: Detailed part-by-part settings

2. Information Storage and Usage

  • Storage Location: Chrome browser only (localStorage/syncStorage)
  • Purpose: Improving music prompt generation quality
  • Encryption: API keys stored with Base64 encoding
  • External Communication: OpenAI API for prompt generation only

3. Third-Party Sharing

Collected information is not shared with third parties. OpenAI API communication is used solely for music prompt generation purposes.

4. Data Deletion

  • Complete Deletion: Uninstall the extension
  • Manual Deletion: Clear storage from Chrome settings
  • Auto Deletion: "Clear Data" button in settings

5. Security

API keys are encrypted and stored only within the browser. They are not transmitted to external servers.
